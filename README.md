# BlackRaven (BLKR)

BlackRaven is an open-source blockchain project forked from Ravencoin.

It is designed for the easy issuance and peer-to-peer transfer of digital assets (tokens, NFTs, and real-world representations) on a fast, secure, and decentralized network.

### Key Features
- **Max Supply**: 32,000,000,000 BLKR
- **Block Time**: ~1 minute
- **Initial Block Reward**: 5,000 BLKR (100% to miners)
- **Pure fair launch** – No pre-mine, no ICO, no developer allocation
- Full support for asset creation and management (inherited and preserved from Ravencoin)
- GPU-friendly Proof-of-Work mining (KAWPOW-based)
- Built for users who want uncensorable, permissionless asset issuance

BlackRaven aims to provide a lightweight, miner-friendly blockchain where anyone can create and transfer custom assets efficiently and securely.
